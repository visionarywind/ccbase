# Information
- CMake multi-module project
- Tests via `Catch2` and `GTest`

## Description 
Base project for learning c++

## Structure
- application
  main function
- common
  common lib
- gtests
- test
- libs
- tools

## Build
```bash
./build.sh --clean
```

Run tests:
```bash
./run_tests.sh
```

## Run application

```bash
./build/application/app
```
